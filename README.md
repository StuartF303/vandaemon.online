# Vandaemon Marketing Website

Official marketing website for the vandaemon project.

## About

This static website showcases the vandaemon project, provides access to a live demo, and guides developers to the GitHub repository and documentation.

## Features

- Responsive design (mobile-first)
- Fast performance (Lighthouse 90+ scores)
- Accessible (WCAG 2.1 AA compliant)
- Google Analytics 4 integration
- Deployed via GitHub Pages
